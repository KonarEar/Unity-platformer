# Unity-platformer

## About the project
Unity-platformer is a small 2D platformer game developed with Unity as a personal project.

The project was created to experiment with gameplay mechanics, level logic, and object interactions inside Unity.  
It represents one of my early attempts to build a complete game from scratch.

The focus of this project was:
- implementing core gameplay systems
- handling player movement and collisions
- creating trap logic and interactive elements
- managing scenes and level transitions

---

## Project status
Prototype / early-stage project

- Code is functional but not fully optimized
- Some logic can be improved and refactored
- Architecture is relatively simple
- Visual assets are minimal

This project demonstrates practical experience with Unity workflow and C# scripting.

---

## Features
- 2D platformer gameplay
- Multiple levels
- Player movement and collision handling
- Traps (spikes, saws, moving platforms)
- Basic animations
- Scene transitions and level management
- Simple UI (main menu and game over logic)

## Technologies
- Unity
- C#
